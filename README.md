# walusiki
Web app security testing with python and mutillidae.

Demonstrating on how to use python to perform web application security testing on Mutillidae, a vulnerable web app project of OWASP.
Presently, most of the arguments are hardcoded and the script's simplicity tends to help people new to coding understand how the code works. 

The Walusiki project aims to become a web application security testing framework.

Be a part of the project, let us grow together along with walusiki.
